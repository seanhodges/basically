## What you'll find there

- **Help &amp; support** — stuck on a dialect quirk, an emulator, or getting a
  program onto real hardware? Ask away.
- **Show and tell** — share the games and programs you're writing, on the
  emulator or the real machine.
- **Machines &amp; dialects** — chat about the classic micros Basically supports
  (and the ones people would like it to).
- **Development** — updates on new features and dialects, and a place to float
  ideas before opening a [GitHub issue](https://github.com/seanhodges/basically/issues).

Prefer GitHub? Bug reports and feature requests are always welcome on the
[issue tracker](https://github.com/seanhodges/basically/issues), and code
contributions via the [contributing guide](/contributing/contributing).
